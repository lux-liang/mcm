\subsection{Preprocessing}

\textbf{O*NET}: Task importance scores normalized ($\sum w_k = 1$); physical tasks flagged via Work Context; AI exposure rated on 5-point scale based on cognitive complexity, data availability, and physical requirements.

\textbf{BLS}: SOC code revisions (2010, 2018) reconciled via crosswalks; May reference period used consistently.

\textbf{IPEDS}: CIP-SOC crosswalk applied; completions adjusted by graduation rates.

\textbf{Quality Assurance}: Cross-validation against expert surveys; temporal consistency checks; outlier detection ($>$20\% YoY flagged); multiple imputation for missing data ($<$5\%).
